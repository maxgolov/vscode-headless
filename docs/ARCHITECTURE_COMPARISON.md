## Deployment Comparison

### **vsc-remote-mcp Deployment**

```bash
# Step 1: Clone or install
git clone https://github.com/ruvnet/vsc-remote-mcp
cd vsc-remote-mcp

# Step 2: Configure Docker Compose
cat > docker-compose.yml << EOF
services:
  vscode-instance:
    build: ./docker
    ports:
      - "8080:8080"
    volumes:
      - ./workspace:/home/coder/workspace
    environment:
      - PASSWORD=secure-password
EOF

# Step 3: Deploy
docker-compose up -d

# Step 4: Start MCP server
npx vsc-remote start --mode websocket --port 3001

# Result: VSCode running in container, accessible via browser
# URL: http://localhost:8080
```
